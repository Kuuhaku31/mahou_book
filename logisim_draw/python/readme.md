# Logisim Draw

### 参数

| 参数      | 功能                      |
| --------- | ------------------------- |
| -m        | 指定模式                  |
| -t        | 指定目标 circ 文件地址    |
| -l        | 指定目标 circuit 标签名称 |
| -p        | 指定待加载图像地址        |
| -h --help | 打印帮助                  |

### 模式

| 模式 | 功能                                  | 此模式下有效参数 |
| ---- | ------------------------------------- | ---------------- |
| del  | 删除指定目标 circuit 标签下的所有像素 | -t -l            |
| add  | 将图片添加到目标 circ 文件            | -t -l -p         |
